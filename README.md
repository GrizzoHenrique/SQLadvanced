# Projeto Módulo 26 do curso de ciêntista de dados da EBAC
**Este projeto tem como objetivo utilizar as operações avançadas do SQL utilizando o SQLITE dentro de um ambiente python e também a prática da criação de um dashboard interativa no Power BI**

## Etapas do projeto
* Exploração da base - Visualizar as tabelas de clientes e transações
* Integração dos dados - Realização do JOIN SQL utilizando a coluna de ID_CLIENTE
* Consolidação dos dados - Criação de uma tabela unificada contendo os dados de clientes e transações
* Exportação do dados - Exportação dos dados em um arquivo .csv para a utilização na criação de dashboards
